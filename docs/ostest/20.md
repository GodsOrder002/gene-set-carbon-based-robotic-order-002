有，而且是资料库里**共识级**的现象。简单给你列几个典型样本：

1. **印尼 1997–1998 亚洲金融危机 → 苏哈托下台**

   * 97 年亚洲金融危机后，印尼卢比暴跌、银行系统和企业债务爆雷，经济直接塌了。
   * 失业、物价暴涨引发大规模骚乱、排华暴力和全国学生运动，最后把统治 32 年的苏哈托政权一起掀翻，进入所谓 *Reformasi*（改革时期），整个执政 OS 换了一套。

2. **阿根廷 2001–2002 债务 / 银行业危机 → 总统和政府垮掉**

   * 长期经济衰退 + 货币制度崩盘 + 国债违约，触发 2001 年 12 月的全国骚乱。
   * 当时总统德拉鲁亚在严重骚乱中宣布紧急状态，最后直升机逃离总统府，被迫辞职，之后几天内连续换了好几个临时总统，原来的政治组合直接被冲掉。

3. **法国 18 世纪末王室财政危机 → 法国大革命 → 君主制被推翻**

   * 18 世纪法国长期打仗+税制烂，国家财政长期亏空，到 1780 年代已经接近破产。
   * 1789 年前后同时爆发财政、经济、政治危机，王室既加不上税又借不到钱，被迫召开三级会议，最后演变成法国大革命，旧王朝（旧制度）整体被推翻。

你要的那个模式，用你自己的话讲就是：

> **金融 / 财政系统崩到一个阈值 → 扛系统的人群不愿再续命 → 连带把原来的政府 / 政治 OS 一起换掉。**

这种“金融崩盘 + 政权崩盘绑在一起”的样本，在可检索历史里是**确实存在**的。
